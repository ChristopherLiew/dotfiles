# Chris' Dot Files

My collection of dotfiles on macOS as managed by chezmoi :)

## Dependencies

1. ```git```
2. ```chezmoi```
3. ```homebrew```
4. ```fish```
5. ```starship```
6. ```pip```

## Usage

Pull from Github and run the ```install.sh``` script

```shell
git pull https://github.com/ChristopherLiew/dotfiles.git
bash ./install.sh
```
